# 칼리온 내부 온실 견적 관리자

이 폴더는 웹 공개용이 아닌 내부 검토용 정적 HTML 도구입니다.

## 실행

로컬에서 아래 파일을 브라우저로 직접 열어 사용합니다.

```text
internal/quote-admin/index.html
```

또는 프로젝트 루트에서 간단한 정적 서버를 띄운 뒤 접속합니다.

```bash
python3 -m http.server 4181
```

```text
http://127.0.0.1:4181/internal/quote-admin/index.html
```

## 포함 기능

- 상세견적 문의 이메일의 `quote_payload` JSON 불러오기
- 고객/문의 정보 입력
- 표준 온실 모델 선택
- 폭, 길이, 측고, 동수 입력
- 피복, 자동개폐, 커튼, 스크린, 관수 옵션 조정
- 파이프, 피복, 조리개, 기초, 모터, 커튼 단가 조정
- 노무비, 마진, 예비비, VAT 조정
- 사진 기반 물량 항목 안내도
- 원가계산서와 물량내역서 자동 산출
- 텍스트 복사와 출력 기능

## 권장 업무 흐름

1. 고객이 공개 견적기에서 상세견적 문의를 제출합니다.
2. 담당자는 이메일로 받은 `quote_payload` JSON을 복사합니다.
3. 내부 관리자 페이지의 `고객 요청 JSON 불러오기`에 붙여넣고 `고객 요청 불러오기`를 누릅니다.
4. 고객 조건이 자동 반영된 상태에서 단가, 노무비, 마진, 예비비, VAT를 조정합니다.
5. 출력 또는 텍스트 복사 기능으로 고객 전달용 견적 초안을 재발급합니다.

## 참고

이 HTML은 `public/` 밖에 있어 Vercel 정적 공개 경로로 직접 노출되지 않습니다.
사용자 공개용 견적기는 `public/quote-tool/index.html`입니다.
