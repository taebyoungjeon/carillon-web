견적 조건

시설 형태

먼저 단동형인지 연동형인지 선택합니다. 선택하면 3D 화면이 바로 바뀌고 다음 단계로 이동합니다.

시설 폭

한 동의 폭을 입력합니다. 연동형은 한 스팬 폭 기준입니다.

시설 길이

실제 시공하려는 온실의 길이를 입력합니다.

측고

측고가 높아지면 골조, 피복, 브레싱, 시공비가 함께 증가합니다.

동수/연동 수

단동은 동수, 연동은 스팬 수 기준입니다.

내재해 기준

요구 적설심과 풍속을 넣으면 가까운 표준 규격을 다시 찾습니다.

작목/용도

작목은 이후 커튼, 스크린, 관수 옵션 추천 기준으로 사용합니다.

피복 비닐

현재 상세 규격 전에는 장수비닐을 기본 임의 사양으로 둡니다. PO필름 선택 시 피복비를 임의 보정합니다.

MESH 환경제어

제어 채널과 센서 포인트를 입력하면 환경제어 HW와 구독료가 산출됩니다.

추가 요청

필요한 추가 사양을 선택합니다. 선택값은 상세내역의 물량과 금액에 반영됩니다.

선택 조건

단동형 7m x 97m

입력 조건에 맞춰 표준 규격과 공사비 기준을 실시간으로 매칭합니다.

추천 모델 스펙

견적 구성

이 화면은 견적 상담을 시작하기 위한 참고 산출입니다. 최종 견적은 현장 실사, 지역별 적설/풍속, 지반, 전기 인입, 배선 거리, 보온/스크린 사양, 정부지원사업 조건 확인 후 확정됩니다.
상세견적 문의하기

이메일, 핸드폰 번호, 기타 요구사항, 상담 희망 여부를 모두 입력해야 상세견적 내역을 볼 수 있습니다. 제출된 정보와 견적 조건은 칼리온 담당자에게 전송됩니다.
